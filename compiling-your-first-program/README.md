# Disabling compiler extensions

```
-pedantic-errors
```

# Enabling All Warnings

```
-Wall
-Weffc++
-Wextra
-Wconversion
-Wsign-conversion
```
# Treat Warnings as Errors

```
-Werror
```

# Set CPP Standard

```
-std=c++17
```


# All Ready to Set to tasks.json (debug build):

```
"-pedantic-errors",
"-Wall",
"-Weffc++",
"-Wextra",
"-Wconversion",
"-Wsign-conversion",
"-Werror",
"-std=c++20",
```
